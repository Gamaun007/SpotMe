## Interaction Patterns

### Trainee UX Priorities

1. **Minimal taps** — Core actions require 1-2 taps maximum
2. **Large touch targets** — Minimum 44pt, prefer 56pt+ for primary actions
3. **Clear feedback** — Haptic + visual confirmation on every action
4. **No ambiguity** — Current state always obvious (what exercise, which set, what to do next)
5. **Forgiving** — Easy undo, no destructive single-tap actions

### Trainer UX Priorities

1. **Information density** — See trainee's full session at a glance
2. **Quick actions** — Log sets rapidly without navigating away
3. **Real-time clarity** — Live data updates are visually distinct
